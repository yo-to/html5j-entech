【公開データ】第1回勉強会 セッション１
======================
### ▼データについて
2014/2/27（木）に開催された __「html5j エンタメ技術部 第1回勉強会 〜 Web技術者のためのArduino入門 〜」__ の __「セッション１：目指せMaker！初心者がArduinoにチャレンジしてみた」__ で、デモに用いたプログラムです。  

Arduinoで下記のようなことを行うためのプログラムになります。
* LEDを点滅させる
* LEDの明るさを変える
* 光センサーを使う
* スピーカーを使う
